Steganography App

By Jason, Deeyaa, Samuel

This app was created for Professor Das' CS-322 Web Programming Course. The app uses LSB or least significant bit steganography to obfuscate data using images as a medium. To use the app, start by uploading an image, a message, and a pgp key to the encoder page and get your encoded image. To decode an image, provide the image and the pgp key on the decoder page to get a message as a result.
